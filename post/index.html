  • Completely uninstall Istio

    2021-02-13

    If you want to uninstall istio on your Kubernetes cluster, Delete istio-system namespace is not enough to uninstall istio. Need to delete CRDs too. Can be completely uninstall follow this command. If you installed using istioctl and demo profile. istioctl x uninstall --purge istioctl x uninstall profile=default
